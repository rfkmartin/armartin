# Adrian R. Martin Website

This project is a personal website for author Adrian R. Martin, showcasing his literary works, biography, and contact information for purchasing his books. The website is designed to provide an engaging experience for readers and fans, allowing them to explore Adrian's books and learn more about his background and writing style.

## Project Structure

The project is organized as follows:

```
adrian-r-martin-website
├── public
│   ├── index.html        # Main entry point of the website
│   ├── books.html       # List of the author's books with cover images and blurbs
│   ├── bio.html         # Artist's biography
│   ├── contact.html      # Contact information for purchasing books
│   └── css
│       └── styles.css    # CSS styles for the website
├── src
│   ├── data
│   │   ├── books.json    # Data about the author's books
│   │   └── author.json    # Data about the author
│   └── scripts
│       └── main.js       # JavaScript for interactivity
└── README.md             # Project documentation
```

## Setup Instructions

1. Clone the repository to your local machine.
2. Navigate to the project directory.
3. Open the `public/index.html` file in a web browser to view the website.

## Features

- **Author Blurb**: A brief introduction to Adrian R. Martin on the homepage.
- **Books Section**: A dedicated page displaying all of Adrian's books, complete with cover images and descriptions.
- **Biography**: A page detailing Adrian's background, achievements, and writing style.
- **Contact Information**: A page providing details on how to purchase books and contact the author.
